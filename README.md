# 🎤 AI Voice Translator (Real-Time)

A real-time voice translator that listens to speech, converts it into text, translates it into another language, and speaks the result instantly.

---

## 🚀 Features

* 🎤 Live voice input from microphone
* ✍️ Speech converted to text in real-time
* 🌍 Instant translation between multiple languages
* 🔊 Audio output of translated text
* ⚡ Smooth, non-blocking UI using threading
* 🖥️ Simple desktop interface

---

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* Deep Translator
* Tkinter
* Threading & Queue system

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/ak-Amaan/AI-Voice-Translator.git
cd AI-Voice-Translator
```

---

### 2. Create virtual environment

```bash
python -m venv env
source env/bin/activate   # macOS/Linux
env\Scripts\activate      # Windows
```

---

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python app.py
```

---

## 🎮 How It Works

1. Click **Start Translation**
2. Speak into your microphone
3. View recognized text in real-time
4. Get translated output instantly
5. Hear the translated speech

---

## ⚙️ Configuration

You can modify inside `app.py`:

* Input and output languages
* Microphone sensitivity
* Phrase recording duration
* Queue size for processing
